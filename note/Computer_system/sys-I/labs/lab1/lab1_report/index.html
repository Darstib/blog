    复合多路选择器及语法分析比较 10% ¶

    我选择使用 index 索引法，因为给我的逻辑很直观，I[3:0] 就是 4 根电线束，每束有 32 根线罢了，而 S[1:0] 的值恰好就是它所选取的电线束的 index

    在这里，index 索引即免去了 if-else 或 case 语法可能会漏掉某种情况的风险，又不用像 ?: 语法那样嵌套使用可读性降低
