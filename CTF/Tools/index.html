OSINT¶

Question

什么是 OSINT ？

  • Digital-Privacy

    • 一个巨大的开源信息站点收集项目

  • 谷歌地图

  • 百度地图
  • 高德地图
  • saucenao
  • GeoSpy
    • 根据图片使用 AI 寻找地点（当然，不是很准，但是他的描述中可能会有你本来没注意到的地方）

    • suncalc
    • 时间→位置互相估计
  • 飞机航班信息
    • -flightaware
    • -flightradar24
    • -adsbexchange
    • 估计方向，位置，时间等
  • 风景信息→Yandex 搜索

  • 天气信息、云层信息等

  • web archive

    • 网页存档

  • 网络空间搜索

  • codesearch
  • sourcegraph

简单编解码¶

  • CyberChef
    • 赛博厨子
  • ciphey
    • 自称快于 CyberChef，且捕获类似于 flag{content} 的结果（如果实际的 flag 比较奇怪，或者有一层混淆之类的，可能导致无法解出）
    • 基本使用：
      • ciphey -t "encode_flag"
      • ciphey -f flag.txt
  • python-codext
    • 能在 python 中使用，也能在命令行使用
  • CaptfEncoder
  • Tupper's self-referential formula
    • tuppers-formula
  • Zero-Width Characters
    • zero-width-web
  • -basecrack <- base 系列爆破
    • -base64decode 还支持文件解码
  • 键盘按键代码

  • 新佛曰

  • CTF 常见编码及加解密

  • Crypto CTFs Tricks

Crypto¶

也许分类没那么正确，主要是否需要一些数学知识来区分古典密码和现代密码。

  • practicalcryptography

古典密码学¶

  • quipqiup or SubstitutionBreaker
    • 换位密码
  • Boxentriq
    • Code-Breaking, Cipher and Logic Puzzles Solving Tools
    • cipher identifier
    • caesar-cipher
    • -vigenere-cipher
  • -Vigenère Solver or -Vigenere
    • 维吉尼亚密码，多挺多的，不知道哪个好用一些；密钥长度超过 30 就别想着能直接爆破了……
  • codecs or codext
    • 更全的解码 python 库
  • 不明觉厉系列：
    • -dcode
    • -dencode

现代密码学¶

  • factordb
    • 大数因式分解，支持的数据大小是我所见过最大的；
    • factordb-python 本地运行。
  • OEIS
    • 在线整数序列百科全书（On-Line Encyclopedia of Integer Sequences）
  • proofWiki
    • 数学性质（证明）查找
  • Sage math
    • sagecell online
    • sage document
      • Cryptography part
    • 学习记录
  • GmSSL
    • 国产商用密码开源库
