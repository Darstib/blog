    CTF tools

    CTF 总是在找工具的路上？用过老是不记得在哪了，这里留个备忘；其他可能有用的东西也放这吧。

    • mzy0 <- 综合性网站

    学习平台 ¶

    • Hello-ctf
    • CTF-wiki

    联系 & 赛事平台 ¶

    • ZJUBUS
      • 需要 ZJU 校网
    • NSSCTF
    • BUUCTF
    • bugku
    • crypto 练习
    • Ethernaut 题库闯关题解

    Web¶

    • Githack
    • DNS lookup
    • regex101
      • 可以 debug 看到详细的正则匹配过程，然后利用匹配步数限制来绕过检测

    MISC¶

    图片类 ¶

    • Aperi'Solve
      • 图片一把梭
    • Forensically
      • 图像取证分析
    • blind_watermark
    • steghide
      • steghide 一篇介绍
    • hex
      • 在线十六进制编辑器

    音频类 ¶

    • slienteye
    • audacity

    寻地类 ¶

    • suncalc
      • 影子 => 地点
    • GeoSpy
      • 根据图片智能寻找地点（当然，不是很准）

    流量分析 ¶

    • wireshark

    区块链与以太坊 ¶

    • remix
      • solidity 在线编辑平台
    • ethernaut
      • 著名以太坊智能合约入门题目集
      • Youtube 上的讲解
      • 我的一些代码
    • powfaucet
      • sepolia 中 ETH 的获取源
    • Ethereum Unit Converter

    其他 ¶

    • 太阳角度、阴影长度等太阳相关
      • 时间→位置互相估计
      • suncalc.org
      • sunearthtools.com
    • 飞机航班信息
      • 估计方向，位置，时间等
      • flightaware.com
      • flightradar24.com
      • adsbexchange.com
    • 风景信息→Yandex 搜索

    • 天气信息、云层信息等

    • 网络空间搜索

      • 钟馗之眼
      • fofa
      • shodan
    • codesearch
    • sourcegraph
    • VirusTotal
      • 文档、url 等病毒检测

    Crypto¶

    • CyberChef
      • 编码神中神
    • cipher identifier
    • ciphey
    • quipqiup or SubstitutionBreaker
      • 换位密码
    • factordb
      • 大数因式分解，支持的数据大小是我所见过最大的
    • codext
      • 更全的解码 python 库
    • boxentriq <- 支持小范围爆破
      • caesar-cipher
      • vigenere-cipher
    • sagecell 在线
      • sage 文档
    • UU 在线工具
      • 文本文件合并
    • basecrack <- base 系列爆破
      • base64decode 还支持文件解码
    • bugku
    • 不明觉厉系列：
      • dcode
      • dencode
